2. Create `sidecar/src/main.ts`:
   - Import `readline` from `node:readline`
   - Import `handleEcho` from `./handlers/echo`
   - Import `Request` type from `./protocol`
   - Create readline interface on `process.stdin`
   - Listen for `process.stdin` `end` event -> `process.exit(0)` (SIDE-05)
   - On each line:
     - Try to JSON.parse the line
     - On parse failure: write error response `{"id":null,"error":{"code":"PARSE_ERROR","message":"..."}}\n` to stdout
     - On success: dispatch based on `msg.method`:
       - "echo" -> `handleEcho(msg.params)`
       - default -> error response `{"id":msg.id,"error":{"code":"UNKNOWN_METHOD","message":"Unknown method: ..."}}`
     - Write response as `JSON.stringify({id: msg.id, result}) + "\n"` to stdout
   - Create a `log()` helper that uses `console.error()` -- NEVER `console.log()`
   - Log startup message to stderr on init: `log("sidecar started, pid:", process.pid)`
